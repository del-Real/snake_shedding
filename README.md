# snake_shedding

### Made with RayLib C++

![](./demo.gif)

### Controls

| Keys             | Action           |
|------------------|------------------|
| WASD/Arrow keys  | Move              |
| G                | Show/hide grid    |
| R                | Restart game      |
| Esc              | Exit game         |
