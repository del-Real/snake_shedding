# snake_shedding

### Made with RayLib

![](./demo.gif)


## Controls

| Keys             | Action           |
|------------------|------------------|
| WASD/Arrow keys  | Move              |
| G                | Show/hide grid    |
| R                | Restart game      |
| Esc              | Exit game         |
